### Getting Started

###### Requirements

1. Python 3.9.19
2. Ubuntu 24.04

###### **Installation**

1. Upgrade package

```sh
sudo apt-het update
```

2. Clone the repo and enter repo folder

```sh
git clone https://github.com/appleiphonedddd/NDHU_FL.git
cd NDHU_FL
```
